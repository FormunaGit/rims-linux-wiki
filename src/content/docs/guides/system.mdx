## Terminal Emulators

:::note
A terminal emulator is a software application that allows you to interact with your computer's operating system using text commands.
:::

- ‚≠ê **[kitty](https://sw.kovidgoyal.net/kitty/)** - Fast and feature-rich GPU accelerated terminal
- [WezTerm](https://wezfurlong.org/wezterm/index.html) - Powerful terminal written in Rust
- [Alacritty](https://alacritty.org/) - OpenGL terminal emulator
- [Guake](https://github.com/Guake/guake) - Dropdown terminal for GNOME desktop environment
- [Konsole](https://konsole.kde.org/) - Lightweight terminal emulator for KDE desktop environment
- [yakuake](https://github.com/KDE/yakuake) - Dropdown terminal based on Konsole
- [Hyper](https://hyper.is/) - Electron-based terminal emulator
- [zutty](https://tomscii.sig7.se/zutty/) - High-end terminal for low-end system
